CitySense

AI-Powered Smart City Civic Issue Detection and Dispatch Platform

CitySense is a smart-city grievance redressal system that enables citizens to report civic problems with images and GPS locations while helping authorities resolve cases efficiently through AI automation and real-time dashboards.

Features

Citizen complaint submission with photos and location

AI-generated formal grievance letters

Real-time status tracking

Authority case management and evidence uploads

Admin analytics dashboard

Civic engagement leaderboard

Tech Stack

Frontend: HTML, CSS, JavaScript (Render)
Backend: Firebase Auth, Firestore, Cloud Functions
AI: Google Gemini API

Workflow

User submits report

Data stored in Firestore
